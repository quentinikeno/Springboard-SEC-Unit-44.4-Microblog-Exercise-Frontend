# Microblog Frontend

This is a frontend for an exercise as part of the Springboard Software Engineering Track to practice async thunks with Redux. It is a simple blogging app utilizing:

-   React
-   React Router
-   Redux Toolkit
-   Axios

## Installation

Install using NPM:

```
npm install
npm start
```

## Usage

```
npm start
```
